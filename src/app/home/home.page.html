Corona Virus: Self Isolation Help

What is this tool for?

If you are self isolating due to a suspected Corona Virus infection, and you would like to request some help from a local volunteer group, this tool will help you.

What this tool does not do

This tool cannot help you with emergencies or health problems. If you have an emergency, please dial 999.

Because this tool requests your personal data, you will need to agree to our privacy policy.
